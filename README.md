# Artifex Lab

A Artifex Lab é uma empresa focada no desenvolvimento de soluções digitais, incluindo sites institucionais, landing pages, sistemas web e aplicações personalizadas.

Nosso objetivo é ajudar profissionais e pequenas empresas a fortalecer sua presença digital por meio de tecnologia moderna, design funcional e desenvolvimento sob medida.

## Serviços

* Sites institucionais
* Landing pages
* Sistemas web personalizados
* Aplicações internas para empresas
* Integrações e automações
* Consultoria em tecnologia

## Tecnologias

* Flutter
* Dart
* JavaScript
* TypeScript
* Node.js
* Firebase
* PostgreSQL
* Supabase
* HTML / CSS
